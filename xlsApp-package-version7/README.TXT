xlsApp is a Java application for importing raw xls test cases files in the testopia database.

It requires jdk1.6 or above and the lib folder with jxl.jar, mysql-connector-java.jar ojdbc14.jar and swing-layout-1.0.4.jar


/usr/lib/jvm/java-6-sun-1.6.0.20/bin/java -jar xlsApp.jar


The javadoc is also provided for further extension.
